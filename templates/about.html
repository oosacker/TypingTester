About this project

  • This web app was written by Natsuki Hasegawa in January 2020, for the course SWEN503 during the Master of Software Development degree, Victoria University of Wellington.
  • The backend web server is written in Python using the Flask web framework.
  • The random words are read from a CSV file with 3902 items. This file is stored on the web server. The word list is sent to the JavaScript code in JSON format.
  • The game results are saved into a CSV file on the web server. Because of this, on some web servers (such as Heroku), the file might not be updated and not display the new results.
  • The frontend is written with Bootstrap 4 and jQuery.
  • The fetch API is used to update the high scores.
  • AJAX is used to send the high scores from the JavaScript code to the Flask web server.
